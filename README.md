# 🎓 EduFix AI: Smart Recovery System

EduFix AI is a diagnostic educational tool designed to identify the root causes of student learning gaps. Instead of just flagging a low score, it uses a **Knowledge Dependency Graph** to trace struggles back to foundational concepts.

## 🚀 Overview
If a student struggles with **Fractions**, EduFix analyzes prerequisites like **Division** or **LCM** to determine if the issue is a current topic misunderstanding or a foundational gap.



## ✨ Features
- **Dependency Tracking**: Maps concept hierarchies using Directed Acyclic Graphs (DAGs).
- **Root Cause Identification**: Differentiates between surface-level errors and deep conceptual gaps.
- **Dynamic Recovery**: Generates tailored action items and resource links (Videos/Practice).
- **Interactive Dashboard**: Built with Streamlit for real-time performance simulation.

## 🛠️ Tech Stack
- **Language**: Python
- **Graph Logic**: NetworkX
- **Analysis**: Scikit-learn
- **Frontend**: Streamlit
